Metodologia BEM: Block, Element Modifier. Es una metodologia agil de desarrollo basada en componentes.

Se aplica para reutilizacion de codigo y escalabilidad indexando cada elemento y sub-elemento modificados

Teorias de las medidas: em por defecto para la fuente por navegadores (16px)y se puede establecer en la propiedad de la caja contenedora del elemento html

viewport: capa padre de visualizacion de una ventana web o todo lo que esta disponible en la pantalla del dispositivo. Se usa para inicializar desarrollo como origen en dispositivos moviles mediante la etiqueta meta

Teorias de cajas HTML-CSS

  • En Bloque: usan toda la extension de la pag
  • En Linea: segun el contenido, es decir el texto

Usar de manera eficiente los shorthand

Propiedad a tener en cuenta box-sizing: border-box,content-box

colorizacion: la mas usada: hexadecimal. Usarlo de esta forma garantiza que tus efectos sean independientes del navegador
